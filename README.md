# 🎵 AI Music Creator  
### 多 Agent 协作的智能音乐生成系统  
### Multi-Agent AI Music Generation System  

---

## ✨ 项目简介 | Overview  

**AI Music Creator** 是一个基于多 Agent 架构的音乐生成系统，能够自动完成从旋律生成到音频输出的完整创作流程。  

**AI Music Creator** is a multi-agent AI system that automatically generates music, from melody composition to final audio output.  

系统通过模块化设计，将音乐创作拆分为多个智能 Agent，实现协作式生成，提升创作效率与灵活性。  
The system decomposes music creation into multiple intelligent agents, enabling collaborative generation with improved efficiency and flexibility.  

---

## 🚀 核心功能 | Features  

- 🎼 **旋律生成（Melody Generation）**  
  自动生成基础旋律结构  
  Generate structured melodies automatically  

- 🎹 **和弦编排（Chord Progression）**  
  根据旋律智能匹配和弦  
  Generate chords aligned with melody  

- 🎨 **风格控制（Style Control）**  
  支持不同情绪与风格（Happy / Sad 等）  
  Control music mood and style dynamically  

- 🤖 **多 Agent 协作（Multi-Agent System）**  
  模块化 Agent 协同完成复杂任务  
  Multiple agents collaborate to complete tasks  

- 🔊 **音频生成（Audio Synthesis）**  
  实时生成并播放音频  
  Generate and play audio in real-time  

- 🌐 **可视化界面（Web UI）**  
  基于 Streamlit 的交互界面  
  Interactive UI built with Streamlit
